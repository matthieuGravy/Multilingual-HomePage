Multilingual HomePage est une application Vue3 qui intègre certaines fonctionalités commme.
- utilisation d'une application vue3 de composition
- le multinguisme grâce au fichier .json
- les animations au scroll grâce à la librairie GSAP
- utilisation de composant pour créer une animation infinie
- webpage responsive

Ce modèle est une version non finie et le template et les fonctionalitées peuvent être améliorée.
